DaVinci Configurator Pro 4.2 SP2

New features

  • 'Compare To...' is available for DaVinci Configurator Base if SIP contains valid RTEConfigurator license
  • Reworked 'Compare To...' algorithm and display
  • Import now offers option to use package structure of import file
    MICROSAR Generator
  • New commandline options '-CondGen' and '-noCondGen': Enable/disable conditional generation of files

Fixed issues

  • Trying to remove Containers results in access violation
  • Validation may lead to crash if JumpTo target is not available
  • Evaluating 'active' state of GENy-modules may not work correctly for splitted DPA projects (rework of algorithm)
  • 'Compare To...' of single module configuration does not work (for splitted DPA projects)
  • Saving large project fails due to 'out-of-memory' exception
  • Using 'Compare To...' may lead to 'out-of-memory' exception during save
  • 'Compare To...' does not correctly merge reference values
    MICROSAR Generator
  • Creating converted ECUC ecuc file for several external Generators now correctly gathers all required module configurations
  • Running multiple instances at once fails due to 'incorrect' file locks
  • Converting ecuc file to different AUTOSAR revision removes MODULE-DESCRIPTION-REF
